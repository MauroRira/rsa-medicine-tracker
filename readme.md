# 💊 RSA Medicine Tracker

A command-line inventory management system designed for healthcare facilities (RSA). Built with Python, it helps operators track medication quantities, expiration dates, and daily dosages efficiently.

## 🚀 Features

- Complete CRUD Operations: Add, view, update, and remove medications from inventory
- Expiration Date Tracking: Automatically alerts when medications are expired
- Stock Duration Calculator: Estimates how many days of stock remain
- Data Persistence: Saves all inventory data to a local JSON file
- Error Handling: Robust validation for user inputs (dates, quantities)
- Interactive Menu: User-friendly command-line interface

## 🛠️ Technologies Used

- Python 3.x
- JSON Module (data persistence)
- Datetime Module (expiration calculations)
- Git & GitHub (version control)

## 📖 How to Use

1. Clone the repository:
   git clone https://github.com/MauroRira/rsa-medicine-tracker.git
   cd rsa-medicine-tracker

2. Run the application:
   python "program for rsa.py"

3. Follow the interactive menu:
   - Option 1: Check current inventory status
   - Option 2: Restock existing medications
   - Option 3: Add new medications
   - Option 4: Remove medications
   - Option 5: Save changes and exit

## 💾 Data Storage

All medication data is automatically saved to medicines.json when you exit the program (Option 5).
